This website uses the open source fonts Young Serif and Rokkitt.

The home graphic and banner are derived from the images Buoy / Boei from Flickr user Remko van Dokkum (CC BY 2.0) and Birds On The Wire from Wikimedia user Tomascastelazo (CC BY SA 3.0). Feel free to use either one under CC BY SA 4.0.

The home page and Get Lost are handwritten HTML/CSS. I wrote a simple static site generator in Go for the Snips section.

I self-host this website on a Digital Oceans VM running Nginx on Ubuntu. I use Cloudflare for CDN and domain registration, and the contact form sends emails with Sendgrid.

I am running a simple backend server I wrote in Go using Gin to handle the contact form and Strava authentication. It runs together with the Nginx server using Docker Compose.

If you're interested in self-hosting your own website and have questions, please reach out!
