Get Lost.

Get Lost is a geographic guessing game using your Strava data.
Log in with Strava with the button below. If you have a private profile, check the box first.
You'll be shown the path you took past activity, with no scale or basemap.
Try to remember where in the world it was, and submit your guess.

I have a private Strava profile, or mostly private activities on my public profile.




Your Strava data is sensitive data because it can identify where you live.

This site does not collect your data, and all activity data is sent directly from Strava to your browser.

To confirm I do not save your data, you can view the source code for this project here.

This website uses the open source fonts Young Serif and Rokkitt.

The home page and Get Lost are handwritten HTML/CSS, and I wrote a simple static site generator in Go for the Snips section.

I self-host this website on a Digital Oceans VM running Nginx on Ubuntu. I use Cloudflare for domain registration and CDN, and Sendgrid to deliver the contact form emails.

I am running a simple backend server I wrote in Go to handle the contact form and Strava authentication, connected using an Nginx reverse proxy.
